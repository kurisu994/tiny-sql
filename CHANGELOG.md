# Changelog

本文件记录 tiny-sql 的版本变化，面向使用者说明主要功能和体验变化，不展开内部实现细节。

格式基于 [Keep a Changelog](https://keepachangelog.com/)，遵循[语义化版本](https://semver.org/)。

---

## [Unreleased]

### 新功能

- 已连接的连接右键菜单新增「新建数据库」，支持选择字符集、排序规则并预览 SQL。

### 体验优化

- 新建 / 编辑连接弹窗改为标签页布局，按常规、SSH、SSL 和高级设置分组。
- SSL 模式与证书路径可随连接保存；高级设置新增保持连接、连接 / 读取 / 写入超时、压缩和自动连接选项。
- 检查更新入口移到 macOS 应用菜单，连接列表头部不再放置更新按钮。

## [0.0.1] — 2026-06-30

### 新功能

- 初始桌面客户端能力：支持创建、编辑、复制、删除、测试并打开 MySQL 连接。
- 支持通过多跳 SSH 跳板连接内网 MySQL，并展示连接过程和断开状态。
- 支持首次连接新 SSH 主机时确认指纹，后续连接自动复用已信任记录。
- 支持浏览数据库、表、字段和表数据，并在内置 SQL 编辑器中执行查询。
- 支持取消长时间运行的 SQL，并在执行写操作前进行二次确认。
- 支持连接配置加密保存，避免敏感连接信息明文落盘。
- 支持应用内检查正式版更新、查看版本说明、下载更新并在安装后重启。

### 体验优化

- 连接列表改为更接近桌面客户端的操作方式：单击选中、双击连接、右键菜单管理连接。
- 新建 / 编辑连接、删除确认、SQL 写操作确认统一改为弹窗交互。
- 优化数据结果表格，较大结果集滚动更顺滑，并在结果被截断时给出提示。
- 优化连接拓扑展示，清楚呈现本机、跳板机和 MySQL 的连接路径与状态。
- 更新 tiny-sql 专属应用图标和左上角品牌标识。

### 安全与稳定性

- 增加 SQL 执行护栏，降低误执行多语句、空语句或危险写操作的风险。
- 增加 SSH keepalive 和连接丢失提示，方便定位是哪一跳出现问题。
- 指纹变更会被拒绝，避免连接到不可信的 SSH 主机。
- MySQL 连接默认更适配内网环境，减少因服务端 TLS 配置不一致导致的连接失败。

### 发布准备

- 完善 macOS、Windows、Linux 的桌面应用打包流程。
- 完善 GitHub Release 发布说明、预发布标记和正式版自动更新源生成。
- 修复正式版自动更新清单生成异常，确保 Linux 版本也能进入更新源。
- 补齐发布检查清单和 dogfooding 记录模板，为 v0.1 正式版试用做准备。

### 待验证

- 真实 3 跳 SSH + MySQL dogfooding 仍待完成。
- MySQL 5.7 兼容性仍待真实环境验证。
- 自动更新端到端流程仍需通过真实发布版本验证。
